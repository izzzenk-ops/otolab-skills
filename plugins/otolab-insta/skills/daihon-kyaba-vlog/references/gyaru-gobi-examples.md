# ギャル語尾・再現版（実動画 全30本）

内容は正確なまま、**語尾の伸ばし・柔らかさ（小さいカナ＋伸ばし棒ー／波ダッシュ〜）を音声通りに再現**した版。
伸ばす位置は実際の音声（Whisper＋ギャル語尾プロンプト）から取得。台本を書くときの**表記の粒度・語尾の付け方の見本**にする。

ポイント：
- **「Vlog」は語尾を伸ばして「Vlog〜」** と表記 → 冒頭は「◯◯、〜の1日Vlog〜。」
- **全部の語尾を伸ばさない**。テンション高め・宣言・実況で伸ばし、内省や情報は伸ばさない。
- 伸ばす音は直前の母音に合わせた小さいカナ（ぅ/ぃ/ぉ/ぇ/ぁ）＋「〜」。
- 台本中の「〔話者名〕」は参考発信者本人の自称名のプレースホルダ。実際に台本を書くときは発信者本人の自称名に置き換える。
