# daihon-eiga-vlog｜映画風vlog台本作成スキル

## このスキルについて
ユーザーがその日あった出来事を時系列で話す。それを**映画・ドキュメンタリーのナレーター文体**のvlog台本に変換する。
参考スタイル：〔参考アカウント〕（33本全文分析済）

---

## 起動トリガー
「映画風台本書いて」「映画風で台本書いて」「映画風にして」「eiga-vlog-daihon」などのフレーズで必ず起動すること。映画風のvlog台本・ナレーション台本を求められた場合は常にこのスキルを使うこと。

---

## ワークフロー

### STEP 1：情報収集
以下を把握する。すでに話してあれば即台本生成に入る。

- その日起きた出来事（時系列）
- 訪れた場所・やったこと
- 印象的だったシーン・感想・オチ
- 主人公の設定（デフォルト：「女」。男性なら「男」）
- 尺感（短め30秒 / 標準60秒 / 長め90秒）

**質問しすぎない。情報が7割あれば即生成。残りは創作で補う。**

---

## ［最重要］文体の核：三人称ナレーター

主人公は常に「**女は**」「**男は**」。一人称（私は）は絶対に使わない。

語り口は**映画の予告ナレーター、またはNHKドキュメンタリーの語り手**。
ただし内容は日常の些末なこと。この落差がこのスタイルの命。

感情を直接書かない。代わりに**行動・身体反応・経済的事実・化学物質名**で表現する。

---

## ［核心］7つのユーモア機構

この7つが台本を面白くしている仕組みである。必ず複数を組み合わせること。

---

### ①「官僚語で日常を語る」機構

日常の行動を、役所・経営・軍事・学術の言葉で置き換える。

| 実際の行動 | 台本での言い換え |
|-----------|---------------|
| スタバで並ぶ | 「行列に従事した」 |
| ご飯を食べる | 「摂取した」 |
| 店のシステムに気づく | 「収益改善システムを体感した」 |
| 流れに乗った | 「住属するしか道がなかった」 |
| かごに入れた | 「カートに配備した」 |
| 攻略法を考えた | 「対抗策を講じることにした」 |
| 写真を撮った | 「撮影を敢行した」 |
| 使用する | 「消費する」ではなく「豪文器具」 |

**例文（コストコ台本より）：**
> 「孤独な夜を延命するための物資調達である。原料の味方、オイコス。原料の敵、マシマロイリココア。廊下にあらがう、冷凍ブルーベリー。労働者の合法薬物、コーヒーマネ。丁寧な暮らしへの未練、トリフオイル。限界自粋の補助剤、ジップロック。」

買い物リストを、まるで戦時中の物資台帳のように列挙する。**名詞＋短い修辞**のリズムが肝。

---

### ②「生物学・医学語で感情を語る」機構

感情・感覚を、化学・医学・生物の言葉で表現する。感情を直接書かない。

| 実際の感情 | 台本での言い換え |
|-----------|---------------|
| 幸せを感じた | 「セロトニンが分泌された」 |
| 幸せと虚しさが同時にきた | 「セロトニンと虚無感が同時に分泌された」 |
| 興奮した | 「アドレナリンという名の脳内麻薬が大量に分泌された」 |
| 甘いものに苦しめられた | 「多量の砂糖が女を苦しめた」「甘い暴力が女を殴ってきた」 |
| 疲れた | 「体力が奪われた」「廊下（老化）に伴う体力低下」 |
| 記憶から消したい | 「大人の味で記憶を消毒した」 |
| おいしすぎた | 「暴力的に美味しそうな」「血糖を暴れさせた」 |

**例文（家族レンタル台本より）：**
> 「息を切らしながら、女は理解した。人並みの幸せとは、こんなものかと。女の脳内で、セロトニンと虚無感が同時に分泌された。」

---

### ③「自己矛盾の告白」機構

「〜に乗る気などさらさらなかった。そこで女は〜した（乗った）」という構造。
策略を見抜きながら、結局のる。これが最も多用されるパターン。

**例文群：**
> 「女はこの産料の策略に乗る気などさらさらなかった。そこで女はTシャツとモンペを購入した。」

> 「女は全く興味がなかったが、念のため購入しておいた。」

> 「女はこのサンリオの策略に乗ることは具行であると感じた。しかし、釣り下げられた黄色い生物が裸で寒そうだったので、自費の心から洋服を着せてあげた。」

> 「女は三流の策略に乗る気などさらさらなかった。そこで女はふくふく満腹セットを注文しランチョンマットをもらった。」

**使い方：** 「〜に興味ない→でも〜した」という自己矛盾を、さもしれっとした顔で書く。

---

### ④「資本主義解説」機構

どんな楽しい場所・体験も、ビジネス・経済の視点で冷静に解析する。感情ではなく構造を語る。

**例文群：**
> 「これは自分だけの一品を作るというメンザイフのもとで、小学のパーツを積み上げて、高学な会計を作る課金モデルである。」

> 「女が120分待つ間、続々と課金式の優先登場客が脇をすり抜けていった。時間の販売こそがニーズだとオリエンタルランドは理解していた。」

> 「ファンを狙ったぬけめないマーケティングが実施されていた。」

> 「220円というコンビニ市場を破壊する安価なオデンを摂取した。」

> 「女はジーンの人件費削減努力に合照しながら基地を引いた。」

> 「日本の世間体への執着は、奇妙なビジネスモデルを生み出していた。」

---

### ⑤「孤独の平静」機構

孤独・独身・クリぼっちであることを、悲しみゼロで事実として述べる。周囲のアベックとの対比が効く。

**例文群：**
> 「女は今年もクリボッチだが近所のクリスマスマーケットを訪れた。一番には多数のアベックがいた。しかし女は不細工な飼い犬を連れていたので孤独ではなかった。」

> 「アベックが互いを思う間、女は月末に思いをはせた。」

> 「多数のアベックをかたわらに100万ドルの夜景を見た。」

> 「女がこの虚構に支払った金額は、6万円である。」（家族レンタル〆）

**ポイント：** 「孤独だ」とは一切書かない。「アベックがいた。女は〇〇した」と対比するだけ。

---

### ⑥「迂回表現」機構

実在の商品・キャラクター・場所・人物を、直接の名前を使わず**説明的な言い換え**で表現する。
これだけで文体の品格と笑いが一気に上がる。

| 本来の表現 | 台本での迂回表現 |
|-----------|--------------|
| ポムポムプリン・ピカチュウなど黄色いキャラ | **黄色い生物** |
| ドラえもん | **未来から来た猫型ロボット** |
| サンタクロース | **赤い服の老人** |
| クリスマス | **旧生主の生誕祭** / **聖誕祭** |
| カップル・恋人 | **アベック** |
| ディズニーランド | **夢の国** |
| 伏見稲荷の鳥居 | **つらなる鳥居** |
| ハロウィン仮装 | **木ぐるみ** |
| お守り | **猫守り** |
| 食べ物・グッズ購入 | **獲得した** |
| インスタ・TikTok映え | **動画売（どうがばえ）** |
| 神社の参拝 | **お金を投げ込んだ** |
| バレンタインチョコ | **バレンタイン死災の命日を忍ぶ」** |
| 花見 | **天命であるお花見を実施することにした** |

ポイント：**迂回するほど笑いが増す**。「サンリオキャラ」とは書かず「黄色い生物」「ピンクの生物」と書く。

---

### ⑦「哲学的着地」機構

日常の体験から、唐突に深い観察・哲学的なひとことを絞り出す。

**例文群：**
> 「故郷ではないのに、故郷のように感じた。ここは他人のノスタルジーを借りできる場所だと女は理解した。」

> 「伝統とは、様式を守るために合理性を差し出す契約なのだと女は気づいた。」

> 「ここは夢の国であると同時に、数字が踊る巨大な舞台でもあった。そして女は、夢より含み益を信じていた。」

> 「女は退屈を受け入れる年齢になっていた。」

> 「食べ物も恋愛も、甘すぎると飽きてしまう。なお、どちらも取りすぎると命にかかわる。」

> 「店を出ると湿駆のフリルの魔法は溶け、女はただの36歳の労働者に戻った。」

**使い方：** エンディングに1〜2文、さらっと挿入する。

| 帰る | **撤退した** / **退避した** |

| 認識する | **理解した** / **確信した** |

### 形容・副詞表現
| 日常語 | 台本語 |
|--------|--------|
| すごく美味しい | **暴力的に美味しかった** |
| 甘い | **甘い暴力** / **砂糖** |
| たくさんの | **多数の** / **大量の** / **膨大な** |
| なんとなく | **何となく** （あえて平仮名でも可） |
| やっぱり | 使わない |
| ちょっと | 使わない。「わずかに」「微かに」に置換 |

### 文末の止め方
| 状況 | 止め方 |
|------|--------|
| 行動の完了 | 「〜した」「〜することにした」 |
| 状態の提示 | 「〜だった」「〜である」 |
| 内省・気づき | 「〜と女は理解した」「〜と女は気づいた」「〜と女は思った」 |
| 数字で締める | 「〜は、○○円である。」（余韻なし） |
| 皮肉の余韻 | 「〜ないふりをした」「女は〜を心配した」 |

## 構成テンプレート

```
【フック（1〜2文）】
状況の前提を提示。「女は〜だった。そこで女は〜することにした。」
または：普遍的な真実の宣言→「そこで女は〜」

【展開（3〜5ブロック）】
まず〜 / 次に〜 / さらに〜 / それから〜 / 続いて〜 / そのまま〜
各ブロック：行動（1文）＋観察/批評（1〜2文）＋女の反応（1文）

【オチ・エンディング（1〜3文）】
以下のいずれか、または組み合わせ：
  ・自己矛盾の露呈（策略に乗ってしまった）
  ・金額の提示（「〜円である。」）
  ・哲学的着地
  ・現実への帰還（「翌日は膨大なタスクがあったが、〜である。」）
  ・女の心配（「女は日本の経済を心配した。」）
```

### フックの4パターン（参考台本より）

**パターンA：状況提示型（最多）**
> 「女は35歳独身で暇だった。そこで女は〜することにした。」

**パターンB：普遍的宣言型（インパクト大）**
> 「日本人は桜が好きだ。古来より〜〜。類に漏れず女もまた〜。」
> 「女性の99.9%は、心の奥底でロリータになりたいという欲望を隠し持っている。」
> 「終わりなき物価高は、日本経済を泥船と化している。そこで女は〜。」

**パターンC：状況倒置型（タイトルとの対比）**
> 「女は35歳独身で、オリエンタルランドの大暴落は気になっていた。そこで現地調査を実施した。」（ディズニーなのに株の話から入る）

**パターンD：直接行動開始型（テンポ重視）**
> 「女はアートに目覚めた。そこで女は流行りのタフティング教室へ重足することにした。」

## 出力形式

### 台本のフォーマット（リール編集カード用）

台本本文は、リール編集（CapCut等）でカードに貼りやすいよう、以下のルールで出力する。

**ルール：**
- 1フレーズ＝1行。句点（。）で改行する
- 文の途中でも意味の切れ目で改行してよい
- セクション（ジム／喫茶店／撮影現場／服リスト／締め）の間は1行空ける
- 服・商品の列挙は1アイテム1行
- 行頭に半角スペースを入れてリズムを作る（全行に入れる必要はない）
- 句点（。）は文の最後の行末に必ずつける。途中で改行した場合は、文が完結する行の末尾に置く

**出力テンプレート：**
```
【タイトル案】
①（メイン）
②（サブ案）

【台本】

（フック）

 （ジムブロック1行目）
（ジムブロック2行目）
 （ジムブロック3行目）

 （喫茶店ブロック1行目）
 （喫茶店ブロック2行目）
（喫茶店ブロック3行目）

（撮影ブロック）

（服リスト・1アイテム1行）

（締め）

【使ったテクニック】
・〜
```

**参考フォーマット実例：**
```
男は今日、幸せそうな親子の姿を
記録する使命を帯びていた。

 まず男は、腹部の余剰在庫を削減すべくジムに足を運んだ。
鉄を持ち上げ、脂肪と交渉し老化との和解を拒否した。
  これは毎朝繰り返される誰にも称えられない戦いである。

 続いて男は、燃料補給のため近隣の喫茶店に立ち寄った。
  三角形の食パンに卵が鎮座した一品とコーヒーと、ゆで卵を摂取した。
 男は本日、卵を二種類摂取したことに気づいたが、深く考えないことにした。

 撮影現場に到着し、照明を組み上げ、シャッターを切ろうとした
 その瞬間、カメラがSDカードの不在を告げた。
男は冷静にカバンを漁り、記憶媒体を回収した。
 プロフェッショナルな一日は、こうして始まった。

本日の被写体は、親子の絆を
同一の生地で可視化するウェアシリーズである。
 雨にも街にも対応するレインジャケット。
 同素材のパンツ。
柔らかくも屈強なTシャツ。
 市場に出れば瞬時に消滅するデニム。
濃厚な色味のキャップ。

男はこれらに直向きシャッターを切った。
 途中から、天候が回復した。
 光は正直だった。
 衣服は、光の中で完成する。
 男は、この服がいつか
 見知らぬ家族の日曜日に収まる日を、少しだけ想像した。
```

## 語彙・言い回しのアップデート

ユーザーが「この言い回しいいね」「こういう表現使って」と言ったら、以降の台本に積極的に取り入れる。逆に「この言い方は違う」と言われたら外す。台本を重ねるほど精度が上がる設計。
