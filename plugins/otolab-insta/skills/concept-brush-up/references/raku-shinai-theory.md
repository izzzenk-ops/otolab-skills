# 「楽をしないコンセプト」理論（SAKIYOMI 2026年5月記事より）

出典: SAKIYOMI「2026年、楽して伸びるアカウントはもう厳しい。今から始めるならこの50コンセプト」

コンセプト10案を作るときは、必ずこの理論のレンズを通すこと。

## 理論の4ステップ

1. **伸びているのは「テーマ」ではなく「欲望」**
   - AI資料作成が伸びる → AIが偉いのではなく「面倒な仕事を早く終わらせたい」欲望に刺さっている
   - 炊飯器レシピが伸びる → 「ちゃんと作りたいけど楽したい」欲望
   - 収納グッズが伸びる → 「考えずに一気に整えたい」欲望
   - テーマを真似てもバズらない。刺さっている欲望を見ること。

2. **「楽」系コンテンツはすでに飽和**
   - 簡単レシピ・AI活用術・時短収納・垢抜け法・副業ノウハウは大量にある
   - 今から同じことをやっても、似た投稿の中に埋もれる。

3. **次に目立つのは「楽をしない人」**
   - 面倒なことをやる人／時間がかかることを続ける人／普通の人が避けることに挑戦する人／真似したいけど簡単には真似できない人
   - 視聴者心理：「この人、本当にやってるの？」「自分には無理だけど見たい」「これ友達に送りたい」「発想おかしいけど気になる」
   - こう思われた瞬間、コンテンツはノウハウではなく**物語**になる。
   - 例：「1フォロワーにつき〇〇する」「ストリート〇〇」系の企画も同じ構造。

4. **コンセプト設計＝ジャンル内で「どんな少数派になるか」を決めること**
   - ジャンル選びだけでは足りない。ジャンルの中でどの少数派ポジションを取るかが本体。

## コンセプトを考えるときの3つの問い

1. このジャンルなら、どこで「楽をしない選択」ができるか
2. 普通の人が避ける「面倒なこと」は何か
3. 見ている人が「誰かに教えたくなる場面」はどこか

## 1案の整理フォーマット（記事の型）

- **コンセプト**：どんなアカウントにするか
- **伸びる理由**：なぜ今のSNSで目立つ可能性があるか
- **人に教えたくなる理由**：なぜ保存・共有・会話につながりやすいか

## 勝ち筋の5つの判断材料（ブラッシュアップで使う）

面白いかどうかではなく「自分の場合、どの勝ち筋を選ぶべきか」で決める。

1. 自分の強み
2. 続けられること
3. 届けたい相手
4. 売りたい商品・サービス
5. 今のアカウントの状態

締めの言葉：「自分が続けられる範囲で、他の人が避ける面倒なことを選ぶこと。その面倒さが個性になり、積み重ねが物語になる。」

## 発想パターン7分類（10案のバリエーション出しに使う）

記事の50案は、だいたいこの7パターンに分解できる。10案を作るときは最低5パターンに散らすこと。

| パターン | 中身 | 記事の例 |
|---|---|---|
| ①極端な制限 | 予算・食材・期間・道具を極端に縛る | 1万円で1週間旅／1食材365日／月給20万から19万貯金 |
| ②全部手作業 | 買えば済むものを素材・ゼロから作る | だし・パン・麺を手作り／家具を素材から／おもちゃ手作り／絵本を自作 |
| ③継続実況 | 地味なことを毎日逃げずに続ける姿を見せる | 1日1万歩／100日学習実況／毎日支出記録／1日10回褒める |
| ④実体験検証 | 自分の体・お金・生活を張って検証する | 顔4分割美容液／マッチングアプリ返信率／全貯金500万運用／タイミーレポ |
| ⑤他人に委ねる | 判断を他人・偶然・占いに預けて展開を読めなくする（委ねる相手はフォロワーではなく、家族・友人・街の人などフォロワー0でも手が届く相手にすること） | 地元の人に聞く旅／ばあちゃん恋愛相談／タロットでアプリ／100人に改善点を聞く |
| ⑥逆転 | 手軽なものに手間をかける／常識の逆を行く | カップ麺メガ進化／ミシュラン完全再現／"じゃない収納術" |
| ⑦過程公開 | 素人が提供側になるまでの過程を見せる | 占い師になるまで／セルフネイルをサロン級に／AIでゲーム毎日1個 |

## 具体例の引き出し集

ジャンル別の実例100個が `references/concept-examples.md` にある。10案を作る前に必ず目を通し、ユーザーのジャンルに近い例を発想の足がかりにする。**丸パクリ禁止**。その人のジャンル・強み・数字・売りたいものに置き換えて使う引き出しであって、答えではない。ジャンルが載っていなくても、7パターンを当てはめれば自作できる。
